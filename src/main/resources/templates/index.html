Guía de Uso del Sistema

Activación por Voz

El sistema se activa automáticamente cuando detecta cualquier sonido de voz. No necesita decir nada específico, solo hable.

  • Activación Automática: Hable cualquier palabra para comenzar
  • Instrucciones Claras: El sistema le dará indicaciones precisas
  • Proceso Guiado: Cada paso será explicado claramente
  • Sin Botones: Todo funciona completamente por voz

Cómo Solicitar Medicamentos

Una vez activado, puede solicitar medicamentos usando frases naturales. El sistema extraerá automáticamente el nombre del medicamento.

Ejemplos de frases que puede usar:

"Necesito paracetamol"
"Quiero ibuprofeno"
"Busco aspirina"
"Dame omeprazol"
"Solicito amoxicilina"
"Requiero loratadina"

Proceso de Confirmación

El sistema confirmará su solicitud antes de proceder con la impresión.

  • Para Confirmar: Diga "Sí", "Afirmativo", "Correcto" o "Confirmo"
  • Para Cancelar: Diga "No", "Cancelar", "Negativo" o "Incorrecto"
  • Nueva Consulta: Después de cada operación, puede hacer una nueva solicitud

Qué Hacer Si...

  • No se activa: Hable más fuerte o acérquese al micrófono
  • No encuentra el medicamento: Intente con otro nombre o pronunciación
  • No está disponible: El sistema le informará sobre la falta de stock
  • Error de reconocimiento: El sistema le pedirá que repita

Requisitos Técnicos

  • Navegador: Chrome, Edge o Safari (recomendado)
  • Micrófono: Debe permitir el acceso al micrófono
  • Audio: Asegúrese de que el volumen esté activado
  • Internet: Conexión estable para consultar la base de datos
